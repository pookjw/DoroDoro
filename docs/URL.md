# URL Schemes

## 대상

- DoroDoro (iOS, Mac Catalyst)
- DoroDoroMac (macOS)

## 목록

### 검색 화면 열기

```
dorodoro://?search
```

### 단어 검색

▼ '가로수길' 검색

```
dorodoro://?search=%EA%B0%80%EB%A1%9C%EC%88%98%EA%B8%B8
```

'kelly' 검색

```
dorodoro://?search=kelly
```

### 현재 위치 검색

```
dorodoro://?searchCurrentLocation
```

### 책갈피 화면 열기

```
dorodoro://?bookmarks
```
